    global-font-size

    $global-font-size: 100% !default;

    Description

    Font size attribute applied to html and body. Defaults to 100% so the value is inherited from the user's browser settings.

    Type

    String

    Example

    font-size: $global-font-size;
font-size: var(--global-font-size);

    $breakpoints: (

  'large': ( min-width: 1024px ),
  'xlarge': ( min-width: 1200px ),
  'medium-down': ( max-width: 768px ),
  'large-down': ( max-width: 1024px ),
  'xlarge-down': ( max-width: 1200px ),
) !default;

    Description

    Breakpoints

    Type

    Array
